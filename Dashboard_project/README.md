# Dashboard Project - Iloilo Food Places

## Description
This project is a Python dashboard built using Dash and Plotly.  
It displays information about popular food places in Iloilo City using graphs (bar chart, pie chart, and scatter plot).

## Files
- `dashboard.py` → main Python script for the dashboard
- `data/food_cities.csv` → CSV file containing food places data
- `requirements.txt` → Python libraries needed to run the project
- `README.md` → project description and instructions
